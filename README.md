# TaxPrime V2 — Modern & Clean

Light theme with emerald green accents. Friendly, modern, SaaS-inspired feel.

## Tech Stack
- **Next.js 15** (App Router), **React 19**, **Tailwind CSS 4**, **TypeScript**

## Quick Start
```bash
npm install
npm run dev
# Open http://localhost:3000
```

## Design
- **Theme**: Clean white with emerald (#059669) accents
- **Typography**: Sora (headings) + Outfit (body)
- **Style**: Modern SaaS, friendly, spacious, rounded
